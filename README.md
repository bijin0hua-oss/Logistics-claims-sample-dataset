## Dataset Description

This sample dataset contains processed logistics claim-related data, including key features such as route type, timeliness of fresh produce delivery, insured amount, sender identifier, origin-based compensation ratio, destination-based claim rate (per 10,000 orders), destination compensation ratio, and actual claim amount.

The dataset has been carefully constructed to preserve the statistical distribution, imbalance characteristics, and feature relationships of the original industrial dataset. It is provided to support the reproducibility of experimental results and facilitate further research on risk-aware logistics modeling.

All data have been anonymized and do not contain any personally identifiable or sensitive information.
